# CodeTranspiler

Code Translator (Keyword-Based Programming Language Converter)

This project is a simple keyword-based code translator that converts source code from one programming language to another by replacing its reserved keywords.

The translator uses a keywords.csv file as a dictionary that maps original language keywords to their equivalents in a target programming language. By loading and modifying this CSV file, you can easily adapt the translator to support different programming languages without changing the core logic of the program.

Please note that this project focuses only on keyword replacement. It does not fully translate syntax, grammar rules, or structural differences between programming languages. Additional parsing rules and language-specific logic would be required to achieve a complete and accurate code translation.

The main purpose of this project is to demonstrate a flexible and extensible approach to keyword-level language translation using a configurable dictionary file.
